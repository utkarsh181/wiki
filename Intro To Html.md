# INTRODUCTION TO BASIC HTML
HTML, or HyperText Markup Language, is a markup language used to describe the structure of a web page. It uses a special syntax or notation to organize and give information about the page to the browser. Elements usually have opening and closing tags that surround and give meaning to content.

# HOW TO COMMENT IN HTML FILE
Example:
```
<!-- <p> This is a paragraph </p> -->
```

2. ## THE ANCHOR ELEMENT `<a>`
[MDN Docs For Anchor element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a)
Creates a external or internal hyperlink.

3. ## THE PARAGRAPH ELEMENT `<p>`
[MDN Docs For Paragraph lement](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/p)

4. ## STRONG IMPORTANCE ELEMENT `<strong>`
[MDN Docs For Strong Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/strong)

5. ## HEADING ELEMENT `<h1>...<h6>`
[MDN Docs For Heading Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/Heading_Elements)
Creates section heading.

6. ## EMPHASIS ELEMENT `<em>`
[MDN Docs For Emphasis Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/em)
Generally , used to make text italics.

7. ## LIST ELEMENT
- ### ORDERED LIST `ol`
[MDN Docs For Ordered List Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/ol)
- ### UNORDERED LIST `ul`
[MDN Docs For Un-Ordered List Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/ul)
- ### LIST ELEMENT `li`
[MDN Docs For List Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/li)
List element actually contains the data of the list.

8. ## TABLE ELEMENT `<table>`
[MDN Docs For Table Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/table)
Table element helps to arrange data in a table.
- ### TABLE HEAD ELEMENT `<th>`
[MDN Docs For Thead Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/thead)
Table Head contain the headers for the tables.
- ### TABLE BODY ELEMENT `<tb>`
[MDN Docs For Tbody Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/tbody)
Table body contains the table excluded by tabel head element.
- ### TABLE ROW ELEMENT `<tr>`
[MDN Docs For Table Row Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/tr)
Table Row element defines rows in table
- ### TABLE DATA ELEMENT `<td`
[MDN Docs For Table Data Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/td)
Table data element is actually the element that stores the data of the table.

9. ## FORM ELEMENT `<form>`
- [MDN Docs For Form Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/form)
Form Has two major  attribute:
1. action - where the form send data to
2. method - what HTTPS method (get/post)
- ### LABEL ELEMENT `<label>`
[MDN Docs For Label Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/label)
Label element  represents a caption for an item in a user interface.
- ### INPUT ELEMENT `<input>`
[MDN Docs For input Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input)
Input Element is used to create interactive controls for web-based forms in order to accept data from the user.
- ### SELECT ELEMENT `<select>`
[MDN Docs For select Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/select)
Select Element is used to make drop down menu in html.

10. ## CONTENT DIVISION ELEMENT `<div>`
[MDN Docs For div Element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/div)
Divsion element is the generic container for flow element. When we want to move content to next line we can use it.
